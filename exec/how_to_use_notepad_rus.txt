Выдели функцию как |y = x ^ 2| потом нажмите ctrl+b для построения графика! 
Вы также можете открыть существующий файл с помощью комбинации ctrl+o!
Ещё Вы можете сохранить проведённые в файле изменения, нажав ctrl+s!
Если Вам нужен пустой файл, просто прожмите ctrl+n и сохраните изменения комбинацией ctrl+shift+s (сохранить как)!

Вы можете поставить два языка на выбор: русский и английский

При написании функции пользуйтесь нижеизложенными операциями:

x + n       ---> сложение x и y
x - n       ---> вычитание n из x
x / n       ---> деление x на n
x * n       ---> умножение x на n 
x ^ n       ---> x в степени n
abs(x)      ---> модуль x

log(x)      ---> ln числа x
log10(x)    ---> lg числа x
logn(x, n)  ---> логарифм x по основанию n

sqrt(x)     ---> квадратный корень x
root(x, n)  ---> корень of x (x в степени (1 / n))

sin(x)      ---> синус x
asin(x)     ---> арксинус x
sinh(x)     ---> гиперболический синус x

cos(x)      ---> косинус x
acos(x)     ---> арккосинус x
cosh(x)     ---> гиперболический косинус x

tan(x)      ---> тангенс x
atan(x)     ---> арктангенс x
tanh(x)     ---> гиперболический тангенс x

cot(x)      ---> котангенс x
